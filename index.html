A bit about me

I have done graduate work in statistics (frequentist & bayesian) and have several years of experience doing statistical & machine learning analyses. This work has predominately been done in the field of Cognitive Neuroscience, working with complex neuroimaging/fMRI datasets, where I have been doing research for the past several years at Dartmouth College. Prior to this I worked in a similar capacity at Princeton University.

I have also worn many hats while building software and doing back-end web development. In these roles, I like using vim, within tmux, from a zshell, via debian (or some derivative thereof). Moreover, I have several years of experience in programming with Python and R (amongst others), in which I enjoy using the many great data analysis, visualization and web programming libraries. In this regard, I gave a talk at the amazing python conference, PyCon, last year.

UPDATE: I'll soon be joining the folks over at Insight for their Data Engineering fellowship!
